# hl_rus
### Данный русификатор бережно сделан мной с нуля и использует комбинированные методы для достижения конечного результата: традиционная озвучка (живым человеком, т.е. мной) + копирование оригинальных голосов с помощью нейросети. Во всех остальных аспектах (включая звуковые эффекты, текстуры, перевод и прочее) нейросети не использовались.
